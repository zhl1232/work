# 下一步计划与路线图

## 🚀 未来功能

### 社交互动

- [x] **用户关注系统**：允许用户关注创作者并在动态流中查看他们的更新（已在移动端实现）。
- [ ] **私信系统**：用户之间用于协作的简单消息系统。
- [ ] **项目协作**：允许与其用户共同创作同一个项目。

### 游戏化

- [ ] **动态徽章**：具有等级的徽章（例如：铜牌、银牌、金牌）。
- [ ] **排行榜筛选**：按时间段筛选排行榜（每周、每月、总榜）。
- [ ] **商店系统**：消耗 XP 解锁个人资料自定义选项（头像框、主题）。

### 内容

- [ ] **视频上传**：支持原生视频上传，而不仅仅是外部链接。
- [ ] **富文本编辑器**：增强项目描述编辑器，提供更多格式选项。

## 🛠 技术债与改进

### 类型安全

- [x] **严格空值检查**：在 `tsconfig.json` 中启用 `strictNullChecks` 并修复产生的错误。
- [x] **Zod 验证**：使用 Zod 对 Supabase 响应和表单输入进行运行时验证。

### 测试

- [ ] **集成测试**：添加从本地 Supabase 实例获取真实（种子）数据的测试。
- [ ] **视觉回归测试**：使用 Playwright 捕捉 UI 回归问题。

### 性能

- [ ] **图片优化**：为用户上传的图片实施高级缓存和尺寸调整策略。
- [ ] **虚拟化**：对长列表（评论、排行榜）使用虚拟化技术以提高渲染性能。

## 📦 部署

- [ ] **CI/CD 流水线**：设置 GitHub Actions 以在推送时自动进行代码检查和测试。
